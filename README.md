Sales Data Analysis Dashboard

Project Overview:
This project is an interactive Power BI dashboard developed to analyze sales data and provide meaningful business insights.
The dashboard helps track payment amounts, billing activities, payment methods, and zone-wise performance through interactive visualizations and KPIs.

Objectives:
Analyze overall sales performance.
Monitor payment trends and transaction volume.
Compare performance across different zones.
Identify popular payment methods.
Provide data-driven insights for decision-making.
Tools Used
Power BI
Power Query
DAX

Dashboard Features:
KPI Cards for key business metrics
Payment Amount Analysis
Payment Type Distribution
Billing Location Analysis
Zone-wise Performance Tracking
Interactive Filters and Navigation Buttons
Detailed Report Page for deeper analysis

Key Insights:
Total payment amount and transaction count are displayed through KPI cards.
Payment trends can be analyzed across different zones.
Billing locations help identify high-performing regions.
Interactive visuals allow users to explore data efficiently.
